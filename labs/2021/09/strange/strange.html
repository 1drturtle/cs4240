What do bin, hex, and oct do?

>>> bin(10)  # Converts the input integer to binary
'0b1010'
>>> hex(10) # Converts the input integer to hexadecimal.
'0xa'
>>> oct(10) # Converts the input integer to octal.
'0o12'

The infix binary operators convert their input integers to binary and perform the binary operations of XOR, OR, and AND respectively.

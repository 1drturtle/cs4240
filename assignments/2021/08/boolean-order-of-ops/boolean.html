Boolean Order of Operations

>>> True or False and False  # and higher than or
True
The operator with the most precedence is not, the second most precedence is and, and the least precedence is or. 
>>> not False or True  # not higher than or
True
>>> not True and False  # not higher than and
False
>>> not (True and False)
True
>>> not False and False
False
>>> not (False and False)
True
